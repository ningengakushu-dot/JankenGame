# 脱初心者を目指す！JavaScript 強化ロードマップ

既にいくつかのステップをクリアし、残りの課題を整理。
難易度順に並べています。

## Level 1：DOM操作とロジックの強化
`innerHTML` を卒業し、より細かく安全な制御を学びます。

### 1. `innerHTML` の使用を減らし、`textContent` と `classList` を使う
- **現状**: 結果を表示する際、HTMLタグごと書き換えている箇所があります。
- **課題**: テキストだけの変更なら `textContent` を使いましょう。また、勝った時に文字を赤くしたい場合などは、CSSクラスを用意しておき、JSで `element.classList.add('red')` のようにクラスを付け外ししてスタイルを変えてみましょう。

## Level 2：データの扱いと機能追加
アプリとして面白くしつつ、データの扱い方を学びます。

### 3. オブジェクトで状態を管理する
- **現状**: `kati`, `make`, `wake` など個別の変数で管理しています。
- **課題**: これらを一つのオブジェクト（例：`const score = { win: 0, lose: 0, ... }`）にまとめてみましょう。

### 4. 「対戦履歴」をリスト表示する
- **課題**: 画面の下の方に `<ul>` タグを用意し、じゃんけんをするたびに `<li>1回目: 勝ち</li>` のように履歴を追加していきましょう。
- **学び**: 配列（Array）の操作と `document.createElement` の使い方を学びます。

### 5. 勝率（%）を表示する
- **課題**: 勝・負・引き分けの数から、「現在の勝率は 〇〇% です」とリアルタイムで計算して表示してください。

## Level 3：ビジュアル演出とキャラクター表現
画像やアニメーションを使い、ゲームらしい体験を作ります。

### 6. テキストから画像への置き換え
- **課題**: 「グー」という文字ではなく、画像（`img`タグ）が表示されるようにしましょう。
- **ヒント**: 既に作成した `hands` オブジェクトを使って、`src` を書き換えます。

### 7. 状況に応じたキャラクターの表情変化
- **課題**: 「待機中」「勝ち」「負け」「あいこ」の状態に合わせて、キャラクター画像を切り替えましょう。
- **ヒント**: `index.html` に追加した `#char-img` の `src` を勝敗のタイミングで変更します。

### 8. シャッフル演出とCSSアニメーション
- **課題**: 結果が出るまで敵の手を高速で切り替える「シャッフル演出」と、結果表示時のエフェクトを実装しましょう。
- **学び**: `setInterval` と `setTimeout` の組み合わせを学びます。

## Level 4：もっとJavaScriptらしく（モダンJS）

### 9. ローカルストレージに保存する
- **課題**: ブラウザを閉じても、勝敗数や連勝記録が消えないように `localStorage` に保存・読み出しする機能を追加しましょう。

### 10. 音を鳴らす
- **課題**: ボタンを押した時や、勝敗が決まった時に効果音を鳴らしてみましょう。

### 11. キーボード操作に対応する
- **課題**: 「1」「2」「3」キーを押してじゃんけんができるようにしましょう。
